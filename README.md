# dotfiles

個人の設定ファイルを管理するリポジトリ。

## セットアップ

```bash
git clone <repo-url> ~/dev/dotfiles
cd ~/dev/dotfiles
chmod +x install.sh
./install.sh
```

## 配置先

| ファイル | シンボリックリンク先 |
| --- | --- |
| `ghostty/config` | `~/Library/Application Support/com.mitchellh.ghostty/config` |
| `claude/settings.json` | `~/.claude/settings.json` |
