# dotfiles

個人の設定ファイルを管理するリポジトリ。

## セットアップ

```bash
chmod +x install.sh
./install.sh
```

## 配置先

| ファイル | シンボリックリンク先 |
| --- | --- |
| `ghostty/config` | `~/Library/Application Support/com.mitchellh.ghostty/config` |
| `claude/settings.json` | `~/.claude/settings.json` |
| `claude/CLAUDE.md` | `~/.claude/CLAUDE.md` |
| `.tmux.conf` | `~/.tmux.conf` |
